## Video Specifications

## Concept Overview

A comprehensive demonstration video showing the Falcon system in action - from monitoring to detection to enforcement. This video tells the story of how the AI protects streets, capturing a real violation and processing it through the system. More detailed and technical than the home page video.

## Visual Style Guide

### Color Palette
| Element | Color | Hex |
|---------|-------|-----|
| Background | Deep Navy Black | #0a0a0f |
| Surface/Cards | Dark Blue-Gray | #1a1a2e |
| Primary Accent | Tech Cyan | #00b4d8 |
| Secondary Accent | Neon Green | #00ff88 |
| Alert/Violation | Bright Red | #ff3b3b |
| Warning | Orange | #f97316 |
| Success/Compliant | Green | #22c55e |
| UI Text | White 70% | rgba(255,255,255,0.7) |

### HUD Design Principles
- **Minimal:** Don't clutter the frame
- **Functional:** Every element has a purpose
- **Animated:** Subtle movements keep it alive
- **Consistent:** Same bracket style, font, spacing throughout
- **Readable:** High contrast, clear typography

### Typography
- **Data readouts:** Monospace font (like SF Mono, JetBrains Mono)
- **Labels:** Clean sans-serif (like Inter, SF Pro)
- **Size:** Readable but not dominant
- **Glow:** Subtle outer glow on key text

### Camera Movement
- Smooth, professional movements throughout
- Gimbal/crane quality - no handheld shake
- Deliberate, slow movements that convey precision
- POV shots should feel stable, like mounted camera

### Lighting
- Night scene with practical light motivation
- Strong contrast between light and shadow
- Cyan accent lighting from the Falcon
- Warm street lamps for color contrast
- Wet pavement reflections for visual interest

### Sound Design Notes
- Ambient city sounds (distant traffic, urban hum)
- Subtle tech sounds for HUD elements
- Soft beep for detection
- Capture "click" sound for evidence capture
- Data transmission: subtle electronic whoosh
- No narration - visual storytelling only
